# Trabajo-de-Gobernanza-de-TI
Proyecto integral de implementación de Service Desk para empresa de telecomunicaciones. Incluye gestión de stakeholders, definición de alcance y KPIs, WBS, cronograma (Gantt), matriz RAM, gestión de riesgos, planificación de recursos y control de costos. Enfoque predictivo con seguimiento de desempeño y mejora continua.
